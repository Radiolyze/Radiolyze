# Viewer Komponenten

## DicomViewer

Rolle:

- DICOM-Canvas
- Tool-Toolbar (Zoom, Pan, Measure, W/L)
- Frame Navigation
- Status Overlay

Status:

- Derzeit Mock UI (kein Cornerstone)
- Vorbereitung fuer DICOMweb und Cornerstone Adapter

## ImageControls

Toolbar fuer Viewer Tools. Aktiviert Tools per Shortcuts.

## SeriesStack

Thumbnail-Stack (frames). Erlaubt schnelles Navigieren.

## ProgressOverlay

Anzeige von:

- ASR Status (idle/listening/processing)
- AI Status (idle/generating/error)
- QA Status (pending/checking/pass/warn/fail)

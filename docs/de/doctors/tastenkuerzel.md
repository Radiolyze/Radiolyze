# Tastenkürzel

Drücken Sie jederzeit in der Anwendung **`?`**, um die In-App-Kürzel-Übersicht zu öffnen.

---

## Allgemein (alle Modi)

| Tastenkürzel | Aktion |
|---|---|
| `Ctrl + S` | Berichtsentwurf speichern |
| `Ctrl + Enter` | Bericht freigeben (öffnet Freigabe-Dialog) |
| `?` | Tastenkürzel-Hilfe anzeigen |

## Sprachdiktat

| Tastenkürzel | Aktion |
|---|---|
| `Ctrl + M` | Mikrofon ein- / ausschalten |

---

## Stack-Viewer (Standard 2D)

### Werkzeuge

| Tastenkürzel | Aktion |
|---|---|
| `Z` | Zoom-Modus aktivieren |
| `P` | Pan-Modus aktivieren |
| `M` | Messwerkzeug (Lineal) aktivieren |
| `R` | Ansicht zurücksetzen |

### Navigation

| Tastenkürzel | Aktion |
|---|---|
| `↑` oder `W` | Vorherige Schicht |
| `↓` oder `S` | Nächste Schicht |
| Mausrad | Durch Schichten scrollen |

### Maussteuerung (Stack)

| Steuerung | Aktion |
|---|---|
| Linksklick + Ziehen | Aktives Werkzeug (Zoom, Pan, Messen) |
| Rechtsklick + Ziehen | Fensterung / Helligkeit anpassen |
| Mausrad | Durch Schichten navigieren |

---

## MPR-Viewer (Multiplanare Rekonstruktion)

Verfügbar für CT- und MR-Studien mit axialen/koronaren/sagittalen Rekonstruktionen.

### Layout

| Tastenkürzel | Aktion |
|---|---|
| `1` | Axiale Ansicht maximieren |
| `2` | Sagittale Ansicht maximieren |
| `3` | Koronare Ansicht maximieren |
| `Esc` | Maximierung beenden (alle drei Ebenen anzeigen) |

### Projektion

| Tastenkürzel | Aktion |
|---|---|
| `M` | MIP (Maximum-Intensitäts-Projektion) ein-/ausschalten |

### Maussteuerung (MPR)

| Steuerung | Aktion |
|---|---|
| Linksklick | Fadenkreuz verschieben |
| Rechtsklick + Ziehen | Pan |
| Mausrad | Zoom |
| `Shift` + Linksklick + Ziehen | Fensterung / Helligkeit |

---

## 3D-Viewer (Volume Rendering / VRT)

Verfügbar für CT-Studien mit volumetrischer Akquisition.

### Presets

| Tastenkürzel | Aktion |
|---|---|
| `1` | Knochen-Preset |
| `2` | Lungen-Preset |
| `3` | Weichteil-Preset |
| `4` | Angiographie-Preset |
| `5` | Muskel/Knochen-Preset |

### Anatomische Ansichten

| Tastenkürzel | Aktion |
|---|---|
| `A` | Anteriore Ansicht |
| `P` | Posteriore Ansicht |
| `L` | Linke Lateralansicht |
| `R` | Rechte Lateralansicht |
| `S` | Superiore (kraniale) Ansicht |
| `I` | Inferiore (kaudale) Ansicht |

### Maussteuerung (3D)

| Steuerung | Aktion |
|---|---|
| Linksklick + Ziehen | 3D-Modell rotieren |
| Rechtsklick + Ziehen | Pan |
| Mausrad | Zoom |
| `Esc` | Kamera auf Standardposition zurücksetzen |

---

!!! note "Tastenkürzel je nach Viewer-Modus"
    Einige Tasten (`M`, `P`, `R`, `S`) haben je nach aktivem Viewer-Modus (Stack, MPR oder 3D) unterschiedliche Funktionen. Der aktive Modus ist in der Viewer-Kopfzeile angezeigt.

# Glossar

**ASR**: Automatic Speech Recognition (Spracherkennung)

**DICOM**: Standard fuer medizinische Bilddaten

**DICOMweb**: HTTP API fuer DICOM Inhalte

**DICOM SR**: Structured Report im DICOM Format

**MPR**: Multi-Planar Reconstruction

**QA**: Quality Assurance

**RAG**: Retrieval Augmented Generation

**EU AI Act**: EU Regulierung fuer KI Systeme

# UI Library

## Stack

- TailwindCSS + shadcn/ui Komponenten
- Design Tokens in `src/index.css`
- Radiology-spezifische Farben (viewer/panel/confidence)

## Guidelines

- Hoher Kontrast (Dark Mode Default)
- Konsistente Spacing Scale
- Fokus-Indikatoren fuer Keyboard Navigation

## Hinweise

Die UI ist auf Langzeit-Nutzung ausgelegt (12h+ Schichten). Zustandsanzeigen
sind bewusst prominent und in Panel-Header integriert.

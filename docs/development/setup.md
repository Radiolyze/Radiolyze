# Setup

## Voraussetzungen

- Node.js 18+
- npm oder pnpm

## Docker (empfohlen)

```bash
docker compose up --build
```

- Frontend: http://localhost:5173
- Backend: http://localhost:8000/api/v1/health

## Install

```bash
npm install
```

## Dev Server

```bash
npm run dev
```

## Build

```bash
npm run build
```

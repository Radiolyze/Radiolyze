# Contributing

## Branching

- Feature Branch pro Arbeitspaket
- Kleine Commits mit klaren Messages

## Code Style

- TypeScript strikt
- Keine neuen Dependencies ohne Review
- UI: Dark Mode kompatibel halten

## Review Checklist

- Funktioniert der Workflow?
- Keyboard Shortcuts getestet?
- Accessibility Fokus sichtbar?
- Keine PHI in Logs?

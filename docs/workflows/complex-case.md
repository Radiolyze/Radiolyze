# Komplexer Fall (CT/MR mit Priors)

## Ziel

Mehrserien-Studie mit Vergleich zu Voruntersuchungen und Guidelines.

## Ablauf

1. **Serie waehlen**: Axial/Coronal/Sagittal.
2. **Messung**: Tool fuer Laesionen (groessenbasiert).
3. **Priors**: Vergleichansicht laden.
4. **Diktat**: Findings mit Vergleichstext.
5. **Guidelines**: Leitlinienhinweise werden angezeigt.
6. **QA**: Warnungen pruefen, ggf. anpassen.
7. **Freigabe**: Approval Dialog, Signatur.

## UI Komponenten

- SeriesList + Viewer Tools
- GuidelinesPanel
- QAChecklist

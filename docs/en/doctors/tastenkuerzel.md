# Keyboard Shortcuts

Press **`?`** at any time in the application to open the in-app shortcut overlay.

---

## General (All Modes)

| Shortcut | Action |
|---|---|
| `Ctrl + S` | Save report draft |
| `Ctrl + Enter` | Approve report (opens approval dialog) |
| `?` | Show keyboard shortcuts help |

## Voice Dictation

| Shortcut | Action |
|---|---|
| `Ctrl + M` | Toggle microphone on / off |

---

## Stack Viewer (Standard 2D)

### Tools

| Shortcut | Action |
|---|---|
| `Z` | Switch to Zoom mode |
| `P` | Switch to Pan mode |
| `M` | Activate measurement (ruler) tool |
| `R` | Reset view to default |

### Navigation

| Shortcut | Action |
|---|---|
| `↑` or `W` | Previous frame / slice |
| `↓` or `S` | Next frame / slice |
| Mouse scroll | Scroll through frames |

### Mouse Controls (Stack)

| Control | Action |
|---|---|
| Left-click + drag | Active tool action (zoom, pan, measure) |
| Right-click + drag | Window / Level adjustment |
| Scroll | Navigate frames |

---

## MPR Viewer (Multi-Planar Reconstruction)

Available for CT and MR studies with axial/coronal/sagittal reconstructions.

### Layout

| Shortcut | Action |
|---|---|
| `1` | Maximise axial view |
| `2` | Maximise sagittal view |
| `3` | Maximise coronal view |
| `Esc` | Exit maximised view (show all three planes) |

### Projection

| Shortcut | Action |
|---|---|
| `M` | Toggle MIP (Maximum Intensity Projection) |

### Mouse Controls (MPR)

| Control | Action |
|---|---|
| Left-click | Move crosshair |
| Right-click + drag | Pan |
| Scroll | Zoom |
| `Shift` + left-click + drag | Window / Level |

---

## 3D Viewer (Volume Rendering / VRT)

Available for CT studies with volumetric acquisition.

### Presets

| Shortcut | Action |
|---|---|
| `1` | Bone preset |
| `2` | Lung preset |
| `3` | Soft tissue preset |
| `4` | Angiography preset |
| `5` | Muscle / Bone preset |

### Anatomical Views

| Shortcut | Action |
|---|---|
| `A` | Anterior view |
| `P` | Posterior view |
| `L` | Left lateral view |
| `R` | Right lateral view |
| `S` | Superior (cranial) view |
| `I` | Inferior (caudal) view |

### Mouse Controls (3D)

| Control | Action |
|---|---|
| Left-click + drag | Rotate model |
| Right-click + drag | Pan |
| Scroll | Zoom |
| `Esc` | Reset camera to default position |

---

!!! note "Shortcuts by viewer mode"
    Some keys (`M`, `P`, `R`, `S`) have different actions depending on whether you are in Stack, MPR, or 3D view. The active viewer mode is shown in the viewer header.

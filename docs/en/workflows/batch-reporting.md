# Batch Reporting (Queue)

## Goal

High volume of reports with short processing time per case.

## Steps

1. **Queue View**: List with priorities.
2. **Quick Review**: Scan findings/impression.
3. **Approval**: Approval dialog per case.
4. **Next**: Automatically load the next report.

## UI Components

- ReportQueue
- ImpressionPanel (Approve)
- Keyboard Shortcuts (Ctrl+Enter)

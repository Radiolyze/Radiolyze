# Complex Case (CT/MR with Priors)

## Goal

Multi-series study with comparison to prior examinations and guidelines.

## Steps

1. **Select Series**: Axial/Coronal/Sagittal.
2. **Measurement**: Tool for lesions (size-based).
3. **Priors**: Load comparison view.
4. **Dictation**: Findings with comparison text.
5. **Guidelines**: Guideline references are displayed.
6. **QA**: Review warnings, adjust if necessary.
7. **Approval**: Approval dialog, signature.

## UI Components

- SeriesList + Viewer Tools
- GuidelinesPanel
- QAChecklist

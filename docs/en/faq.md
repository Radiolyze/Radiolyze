# FAQ

## Is the UI production-ready?

No. The UI is a reference implementation and is not production-ready
(no auth/RBAC, some fallbacks, and open issues).

## Can I run the stack on-prem?

Yes. The architecture is designed for on-prem or private cloud deployment.

## How is EU AI Act compliance achieved?

Through audit logging, human oversight, technical documentation, and monitoring.

## Is Cornerstone already integrated?

Yes. Cornerstone is integrated (Stack Viewer, Tools, W/L Presets); advanced tools are still pending.

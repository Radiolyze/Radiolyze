# UI Library

## Stack

- TailwindCSS + shadcn/ui components
- Design tokens in `src/index.css`
- Radiology-specific colours (viewer/panel/confidence)

## Guidelines

- High contrast (dark mode by default)
- Consistent spacing scale
- Focus indicators for keyboard navigation

## Notes

The UI is designed for extended use (12h+ shifts). Status indicators are
deliberately prominent and integrated into panel headers.

# Reporting Components

## FindingsPanel

- ASR dictation (audio upload + API fallback)
- Confidence bar
- Edit/save mode

## ImpressionPanel

- AI draft display (inference/impression API, optional mock fallback)
- QA status/warnings
- Approval via approval dialog
- DICOM SR export (JSON draft + binary)

## QAChecklist

Collapsible list of QA checks including status and message (API + fallback).

## TemplatesPanel

Displays templates and allows insertion into the report findings.

## GuidelinesPanel

Displays guidelines and warning status per guideline.
